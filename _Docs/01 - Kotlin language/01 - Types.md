# Types

## Primitive data types

- Byte: numeric
- Short: numeric
- Int: numeric
- Long: numeric
- Float: numeric
- Double: numeric
- Char
- Boolean
- String

Byte, Short, Int and Long has a corresponding unsigned:

- UByte: numeric
- UShort: numeric
- UInt: numeric
- ULong: numeric
